# Directives

Questa directory contiene le SOP (Standard Operating Procedures) scritte in Markdown.

Ogni direttiva definisce:
- **Obiettivi**: cosa deve essere raggiunto
- **Input**: dati o informazioni necessarie
- **Strumenti/Script**: quali script in `execution/` utilizzare
- **Output**: risultati attesi
- **Casi limite**: situazioni particolari da gestire

Le direttive sono documenti vivi: vengono aggiornate man mano che si scoprono vincoli, approcci migliori o errori comuni.
